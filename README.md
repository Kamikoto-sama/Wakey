# Wakey
Pet-project for activation and connection to my home PC remotely without having "white IP", using ESP32 and VDS for backend hosting.

## Stack:
### Backend
- .NET (Api and Daemon)
- .NET nanoFramework (Proxy on ESP32)
- WakeOnLan
- ASP.NET and SignalR
- SSH port forwarding
